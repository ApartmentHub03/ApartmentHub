# ApartmentHub

Rental platform for apartment listings in the Netherlands — tenant applications, document uploads, viewing bookings, offer/deal workflows, and CRM automation.

## Tech Stack

- **Frontend:** Next.js 15 (App Router), React 19, Tailwind CSS 4, Redux Toolkit
- **Backend:** Supabase (PostgreSQL, Edge Functions, Storage)
- **Auth:** Phone-based OTP via WhatsApp; Supabase Auth for the CRM
- **Email:** Resend
- **Automation:** n8n Cloud webhooks
